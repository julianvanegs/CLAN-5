# Purchase Registration System in Python

## Description

This project is a simple purchase registration system developed in Python.

The program is designed to store products, calculate totals, and simulate a small purchase process. The system is a basic example used to practice functions, loops, and input validation.

## How it works

1. The program creates a list called `data_base` that stores product information.
2. The user selects an operation from a menu displayed on the screen.
3. The program allows the user to add a new product including its name, price, and quantity.
4. The system calculates the total value of each product by multiplying the price by the quantity.
5. The program stores the product information as a tuple inside the database list.
6. The system displays all stored products and calculates the final purchase total.

## Status

> The project is currently running as a basic purchase management program used for educational purposes.
