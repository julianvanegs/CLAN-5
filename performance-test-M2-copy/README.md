The website was created with the porpose of show more information of The Shattered Riffs

## Technologies
The Three mains technologies used for the web was HTML for the extructure of the page given a more organized method,CSS for style and decoration of the page using colors of the band and JavaScript for functions of Dinamic Buttons Like Navbar in movile
